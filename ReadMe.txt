Often small utility programs make us more productive and save us from repetitive cumbersome work. 
This is a tool box that consists of small applications that I use on my day-to-day .NET development

The first is wex, which opens windows explorer with the current directory displayed from command prompt. 
As time goes on, more applications will be added but now just this one.

The list of feature or applications 

* wex: open windows explorer at the current directory from command line. Just type wex
* Visual Studio 2010 Command Prompt Here: a contextual menu on Windows Explorer. Open visual studio command prompt with the given directory open visual studio command prompt from windows explorer
